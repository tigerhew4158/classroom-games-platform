S001～S007 Vercel 更新包 v16

本版重点修正 S005 幸运转盘：
- 转盘改成真正的正圆圆盘，不再是斜斜的 3D 椭圆盘。
- “转动”按钮固定在圆盘中心。
- 点击“转动”后，整个正圆圆盘会明显旋转。
- 转动结束后会抽出事件，并继续原本事件流程。
- 保留 S005 转盘事件设定、比例选择与题库功能。

上传方式：
1. 解压本 ZIP。
2. 把里面的 games 文件夹覆盖到 GitHub repo 根目录的 games 文件夹。
3. Vercel 重新部署后，用 Ctrl + F5 强制刷新。
4. 建议测试 S005 地址加入 ?lang=zh&v=16。
