# 課堂游戲網站｜白板專用優化版

這個版本針對老師與互動白板使用情境優化。

## 主要功能

1. 老師登入後，只看到自己已授權的遊戲。
2. 每個可用遊戲都有：
   - 白板上課
   - 下載 ZIP
   - 制作
3. 白板上課模式：
   - 中央安全顯示區
   - 無上下捲軸
   - 自動等比縮放
   - 工具列縮小，讓遊戲區更大
4. 離線 ZIP：
   - 老師可下載單一遊戲 ZIP
   - 解壓後直接打開 `index.html`
   - 不需要登入平台也能使用該遊戲
5. 制作區：
   - 老師可先整理課堂標題、科目、題庫
   - 可下載題庫 TXT
   - 正式版可接後端，把題庫寫入遊戲後自動生成專屬 ZIP

## 孤岛抢夺赛显示修正

修正白板播放器中打开「孤岛抢夺赛」时，老师设置框被缩得太小的问题。

原因：
- 孤岛抢夺赛使用全屏设置层和绝对定位地图。
- 原播放器会扫描整个 iframe DOM 来自动计算缩放比例。
- 这类全屏游戏容易被误判尺寸，导致中间设置框异常缩小。

修正：
- 孤岛抢夺赛在播放器中改用固定 16:9 白板画布尺寸。
- 打开该游戏时默认进入 16:9 安全区。
- 老师设置卡片放大为白板适用宽度。


## 孤岛抢夺赛显示修正 v2

进一步修正：
- 孤岛抢夺赛在白板播放器中不再使用「大画布 + transform 缩放」方式。
- 改为直接把 iframe 尺寸设成安全区内的 16:9 或 4:3 响应式大小。
- 这样老师设置面板会按真实白板尺寸排版，不会再缩成中间一小块。
