S001～S007 Vercel 更新包 v2

本包包含 games/ 以下 7 个已确认测试通过的游戏文件夹，并修正 S003 开始游戏无法读取题库的问题。

本次 S003 修正：
- 开始游戏会直接读取题库 textarea，不需要另外套用。
- 题库解析支持中文符号：｜ ／ ： ， ； →。
- 载入范例不再调用旧版未定义 SAMPLE。
- 设定好题库后点击“开始游戏”会正常进入游戏。

上传方式：
1. 下载并解压本 ZIP。
2. 把解压后的 games 文件夹上传到 GitHub repo 根目录。
3. 覆盖原本 games 里对应的 7 个游戏文件夹。
4. Vercel 自动重新部署后，用 Ctrl + F5 强制刷新。

注意：
- 不要只上传 ZIP。
- 这不是完整网站整包；这是 games 文件夹更新包。
- 你的 index.html / app.js / 后台首页不用改，只覆盖 games 即可。
