课堂对应文字对战｜Classroom Text Match Battle

使用方式：
1. 解压 ZIP。
2. 双击 index.html 打开。
3. 老师可选择介面语言：简体中文、English、Bahasa Melayu。
4. 老师可选择 2 组或 3 组。
5. 修改题库后按「套用题库」，再按「开始游戏」。

题库格式：
每一行：提示文字,正确答案
例如：
香蕉,banana
apple,苹果
马来西亚首都,吉隆坡
国旗,Jalur Gemilang

注意：
- 系统不会自动判断英文或中文。
- 左边永远是题目/提示，右边永远是正确答案。
- 答案可以是英文、中文、马来文、数字或短句。
- 支持离线使用，不需要网络。
