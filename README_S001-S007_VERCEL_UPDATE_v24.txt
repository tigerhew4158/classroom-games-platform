S001～S007 Vercel 更新包 v24

本版将 S005 幸运转盘问答重新整理成稳定版，移除之前多层补丁造成的分数重复跳动问题。

S005 v24 修正重点：
- 只保留一套得分逻辑。
- 答对只加一次分。
- 答错只扣一次分或不扣分，依老师设定。
- 答题后会自动换到下一队。
- 分数不会再一直跳动。
- 只保留一套得分板，不会重复出现队伍。
- 当前队伍会黄色高亮。
- 转盘是正圆，可以正常转动。
- 转盘停止后会出现题目。
- 保留转盘事件设定、三语、范例题库、16:9 / 4:3 / 3:4 / 9:16。

上传方式：
1. 解压本 ZIP。
2. 把里面的 games 文件夹覆盖到 GitHub repo 根目录的 games 文件夹。
3. Vercel 重新部署后，用 Ctrl + F5 强制刷新。
4. 建议测试 S005 地址加入 ?lang=zh&v=24。
