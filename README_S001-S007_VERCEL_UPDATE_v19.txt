S001～S007 Vercel 更新包 v19

本版重点修正 S005：
- 修正转盘停止后只弹出“请回答题目”，但没有真正显示题目的问题。
- 不再依赖旧的 showQuestion 函数，改为内建新的题目弹窗渲染。
- 选择题会直接显示题目和选项。
- 开放题会显示“显示答案 / 答对 / 答错”。
- 答题后会更新分数，并出现“继续”按钮。
- 如果题库为空，会自动载入当前语言范例题库。

上传方式：
1. 解压本 ZIP。
2. 把里面的 games 文件夹覆盖到 GitHub repo 根目录的 games 文件夹。
3. Vercel 重新部署后，用 Ctrl + F5 强制刷新。
4. 建议测试 S005 地址加入 ?lang=zh&v=19。
