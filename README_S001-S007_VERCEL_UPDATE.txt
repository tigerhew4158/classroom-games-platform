S001～S007 Vercel 更新包说明

本包只包含 games/ 以下 7 个已确认测试通过的游戏文件夹：

games/memory_match_game
games/true_false_flash_game
games/sentence_order_game
games/treasure_box_game
games/lucky_wheel_quiz_game
games/tug_spell_game
games/whack_a_mole_quiz_game

更新方式：
1. 如果你的 Vercel / GitHub 项目已经有完整网站，只需要把本包里的 games 文件夹上传/覆盖到 GitHub repo 根目录的 games 文件夹。
2. 不需要只上传 zip 到 Vercel；要解压后上传文件夹与文件。
3. 上传后让 Vercel 重新 Deploy。
4. 部署后用 Ctrl + F5 强制刷新网页，避免浏览器缓存旧版本。

注意：
- 这个包不是完整24游戏平台首页包；它是 S001～S007 的 games 文件夹更新包。
- 如果你要完整24游戏平台整包，需要用包含 index.html / app.js / styles.css / games / downloads 的完整项目根目录版本。
